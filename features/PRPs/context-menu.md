# COntext menu

Pracuje na mac os
Do swojej aplikacji chce stworzyc nowa funkcjonalnosc. Gdy nacinse globalnie (aplikacja nie musi byc aktywna)  "control" + "Q" - to ma pojawic sie cos w stylu contextowego menu.
Zalozenie jest takie ze zaznaczam tekst -> klikam control + Q i pojawiaja mi sie opcje ktore sa w commands np. "Przelumacz". Jak klikne jakas z opcji to kopiuje zaznaczony tekst, wkleja sie on do chatu i wysyla. Gdy dostane opdowiedz to wypisuj sie tak jak obecnie, ale gdy dostane juz FINISH i cala odpowiedz to wklejam ja w miesjcu gdzie zaznaczalem tekst. Jezlei okno Alina nie jest aktywna to nie aktywuj go, neich dziej sie to w tle.
Czyli z perspektywy uzytownika dzieis sie to tak. Zaznaczam teskt w dowolnym miejsuc np. na slacku. klilam contorl + Q, wyswietl ami si ecos w stylu menu kontekstowego, wybieram opcje ktore mam zpisane w commands np. "Popraw błędy", gdy dostane odpowiedz to wkleja sie poprawiony tekst.
