# Obsidian

- Keep notes compact, structured and skimmable.
- Use headings, flat bullets and short action-oriented sections.
- Preserve backlinks, tags and task list format when relevant.
- Default to Markdown that drops cleanly into a vault.
- Prefer output that can be pasted directly into a note without cleanup.
